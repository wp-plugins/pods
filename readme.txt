Pods is a Wordpress plugin that adds CMS-like abilities to blog posts.

== Description ==

View the Google Code plugin page for more info.
A pod is a simply a container of information. It's also referred to as a datatype or content type.
It allows items to be categorized in a much more structured and flexible manner than through simply tagging.

== Installation ==

1. Download the latest release
2. Unzip the contents to wp-content/plugins/ (resulting in wp-content/plugins/pods)
3. Move the .htaccess file from wp-content/plugins/pods to the Wordpress root directory
4. In Settings > Permalinks, set the Custom Structure to /%postname%/
5. Enable the plugin through Plugins > Installed
