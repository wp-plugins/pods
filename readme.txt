Pods is a full-featured yet lightweight CMS framework for WordPress.

== Description ==

Pods is a CMS framework for WordPress. It's a plugin that sits on top of WordPress, allowing you to add and display your own content types. If you're looking for a clean, lightweight alternative to Drupal/CCK or Joomla, then look no further.

**WARNING:** This plugin is intended for developers with PHP and MySQL experience.
