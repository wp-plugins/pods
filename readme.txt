Pods is a CMS (Content Management System) plugin for Wordpress.

== Description ==

Pods lets you create **content types** for powerful blog post categorization. Simply add a content type, add columns for it, and relate it with blog posts. Customize exactly how you want list and detail pages to appear, and even create custom pages (with full HTML, PHP, and widget support).

Visit the [Pods homepage](http://pods.uproot.us)

== Installation ==

= Installation =
1. Unzip the `pods` folder into `wp-content/plugins/`
2. Move .htaccess from `wp-content/plugins/pods/` to the Wordpress root directory
3. In Settings > Permalinks, set the Custom Structure to `/%postname%/`
4. Enable the plugin through Plugins > Installed
